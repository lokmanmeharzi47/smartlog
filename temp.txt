Module Prédiction IA6.1  Algorithme WMA (Weighted Moving Average)Algorithme principal de prédiction. Donne plus de poids aux jours récents pour capturer les tendances.WMA = Σ(wᵢ × xᵢ) / Σwᵢ   avec poids [1, 2, 3, 4, 5, 6, 7]Le jour le plus récent a le poids 7, le plus ancien le poids 1ExempleSL-006 — 7 derniers jours : 66,83,87,75,94,74,138 unitésWMA = (66×1+83×2+87×3+75×4+94×5+74×6+138×7) / 28 = 95.5 u/jourPrévision 7j = 95.5 × 7 = 668u  |  Prévision 14j = 95.5 × 14 = 1 337u6.2  Date de ruptureJours_restants = floor(Stock_actuel / WMA_journalier)Date rupture = Aujourd'hui + Jours_restantsExempleSL-006 : Stock=1250, WMA=95.5 → 1250/95.5 = 13 jours → Rupture le 31 mars 20266.3  Taux de confianceCV = σ/μ    Confiance = max(60%, min(97%, (1-CV×0.8)×100%))CV faible = demande stable = confiance élevée6.4  RecommandationsStatutConditionMessage🔴 UrgentJours < 7Commander maintenant🟠 Cette semaine7 ≤ Jours < 14Planifier commande🟢 OKJours ≥ 14Stock suffisant7.
